# Arch Dustro
Arch Dustro (or just "Dustro") is an Arch-Linux based Distro with a custom Desktop Environment (TilDE) made entirely in Rust.
Note: Dustro, and more specifically TilDE, the Desktop Environment are still in development and likely won't be released, or available for a long time.
Ideally the first stable version is to be release before 2028.

# TilDE - The Components
TilDE is made entirely in Rust, and most of its components are made using GTK3 and intended for X11.
https://github.com/ArchDustro/TilDE

## Emdash - Taskbar
Emdash is TilDE's always-visible custom control strip for managing and switching between apps, viewing essential system information and accessing system tools.
Emdash, like the rest of TilDE is made entirely in Rust, for strong memory safety and durability.
Status: Not currently in development yet.
https://github.com/ArchDustro/emdash

## Ellipsis - Application Menu
Made for Emdash - Ellipsis is TilDE's Application Menu. Used for parsing text and have an easy way to get to the apps you need most, Ellipsis sits right there in the Emdash bar.
Status: Not currently in development yet.
https://github.com/ArchDustro/ellipsis

## Underscore - Notifications Daemon
Underscore is the notifications daemon for TilDE. 
Status: Currently in development.
https://github.com/ArchDustro/underscore
